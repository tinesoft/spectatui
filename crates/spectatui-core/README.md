# spectatui-core

Core library for [spectatui](https://crates.io/crates/spectatui) — filesystem watching, Spec-Kit project parsing, GitHub API integration, and shared domain types.

This crate is an internal implementation detail of `spectatui` and does not provide a stable public API. It is published to crates.io solely to satisfy the dependency chain when installing `spectatui` via `cargo install`.
